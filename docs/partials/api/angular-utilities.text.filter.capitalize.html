capitalize
filter in module angular-utilities.text

Description

Allows you to make capital the first letter of words in a given string. Defaults to just transforming the first word, but can be configured to transform all words.

Usage

In HTML Template Binding

{{ capitalize_expression | capitalize[:all] }}

In JavaScript

$filter('capitalize')(input[, all])

Parameters

ParamTypeDetails
inputString

Any string.

all
(optional)
Boolean

Indicates whether to transform all words in the given string.

(default: false)

Returns

String

The transformed string.
