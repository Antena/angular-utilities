isolateScrollingWhen
directive in module angular-utilities.scroll

Description

Isolates the scrolling event within this directive's element on a given condition. This can be used to avoid scroll events from reaching the $document.

Usage

as attribute
<ANY isolate-scrolling-when="{String|Function}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
isolateScrollingWhenStringFunction

Angular expression to watch for. When this expression evaluates to truthy, the scroll event will be isolated within this directive's element. When it evaluates to falsy, the default event bubbling is restored.
