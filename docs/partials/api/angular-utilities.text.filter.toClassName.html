toClassName
filter in module angular-utilities.text

Description

Converts a string into a valid className. Valid chars are '_', '-', a-z, A-Z, 0-9. First char cannot be 0-9 or '-'.

Usage

In HTML Template Binding

{{ toClassName_expression | toClassName[:prefix][:replacement] }}

In JavaScript

$filter('toClassName')(input[, prefix][, replacement])

Parameters

ParamTypeDetails
inputString

Any string.

prefix
(optional)
String

An optional string to use as a prefix for the converted result

(default: '')

replacement
(optional)
String

An optional string to use as a replcement for invalid characters.

(default: '_')

Returns

String

The transformed string.
